# ✨ Vibe Calculator ✨

**My First Vibe Coded UI!**

This is my first project using vibe coding - a super fun way to create beautiful interfaces! Built with the help of Claude Code, this calculator comes in three different flavors.

## 🎨 What's Inside

### 1. **Terminal Calculator** (`calculator.py`)
A colorful command-line calculator with:
- Beautiful ASCII art banner
- Color-coded operations (green for addition, blue for subtraction, etc.)
- Emojis and styled output
- Full error handling

### 2. **Native GUI Calculator** (`calculator_gui.py`)
A desktop application with:
- Dark theme with vibey colors
- Tkinter-based interface
- Modern button styling
- Real-time calculations

### 3. **Web Calculator** (`calculator.html`)
The star of the show! Features:
- Stunning gradient background
- Glowing "Amazing Calculator" title
- Smooth animations and hover effects
- Glassmorphism design
- Full keyboard support
- No dependencies - just open and use!

## 🚀 How to Use

### Terminal Version
```bash
python3 calculator.py
```

### GUI Version (requires tkinter)
```bash
# Install tkinter first (if needed)
sudo apt-get install python3-tk

# Run the GUI
python3 calculator_gui.py
```

### Web Version
Simply open `calculator.html` in your browser - no installation needed!
